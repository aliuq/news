
## 项目说明
网站包含新闻、手机、管理模块，有新闻发布、评论、点赞、查看手机信息、新闻和手机产品操作等功能

## 技术栈
> koa 1.3.0，
vue 1.0.26，
jquery，
mysql

## 数据表结构
#### 数据表结构
> database/create
#### 数据
> admin: 管理员表
database/topics: 文章表
database/details: 文章详情表
database/mobileinfo: 手机信息表
database/params: 手机参数表
database/user: 用户表

## 使用
<pre><code>安装依赖: npm i</code></pre>
<pre><code>运行: npm start</code></pre>
